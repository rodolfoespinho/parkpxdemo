# 🚗 PenichePark
### Smart Digital Parking for Coastal Cities

PenichePark is a **digital parking management prototype** designed to modernize how parking works in coastal and tourist cities.

The platform demonstrates how municipalities can replace or complement traditional parking meters with a **mobile-first, QR-based parking system**, allowing drivers to start parking sessions directly from their smartphones.

This project was developed as a **concept prototype for Peniche, Portugal**, exploring how digital infrastructure can improve urban mobility, reduce operational costs, and simplify the parking experience for residents and visitors.

# 🇵🇹 PenichePark
### Estacionamento Digital Inteligente para Cidades Costeiras

PenichePark é um **protótipo de sistema digital de estacionamento** criado para modernizar a forma como o estacionamento funciona em cidades costeiras e turísticas.

A plataforma demonstra como os municípios podem substituir ou complementar parquímetros tradicionais através de um **sistema de estacionamento baseado em QR codes e acesso móvel**, permitindo aos condutores iniciar sessões de estacionamento diretamente no telemóvel.

Este projeto foi desenvolvido como **prova de conceito para a cidade de Peniche**, explorando como a infraestrutura digital pode melhorar a mobilidade urbana, reduzir custos operacionais e simplificar a experiência de estacionamento para residentes e visitantes.

# 📱 How It Works | Como Funciona

### Step 1
Driver scans a **QR code** placed in a parking zone.

### Step 2
The mobile web app opens automatically.

### Step 3
The user selects:

- parking duration
- vehicle
- payment option

### Step 4
The parking session begins instantly.

---

### Passo 1
O condutor lê um **QR code** colocado na zona de estacionamento.

### Passo 2
A aplicação web abre automaticamente no telemóvel.

### Passo 3
O utilizador seleciona:

- duração do estacionamento
- veículo
- método de pagamento

### Passo 4
A sessão de estacionamento inicia imediatamente.
